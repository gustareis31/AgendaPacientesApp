# AgendaPacientesApp

A simple Android application skeleton.

## Requirements
- [JDK](https://adoptopenjdk.net/) 8 or higher
- [Android SDK](https://developer.android.com/studio) with build tools

## Building
Run the Gradle wrapper to compile the project:

```bash
./gradlew assembleDebug
```

The APK will be located in `app/build/outputs/apk/debug/`.

## Running
Open the project in Android Studio and click **Run**, or install the generated
APK on an emulator or physical device.
